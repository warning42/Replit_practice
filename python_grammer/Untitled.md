```python
# round(실수형데이터, 반올림하고자 하는 위치-1)
a = 0.3 + 0.6   # 0.899999999
print(round(a,4))  # 0.9 -> 소수점 5번째 자리에서 반올림
```

    0.9
    


```python
a, b = 7, 3
```
